Eternal Lands LNX Guild

This is the current home for LNX guild, a group of Linux loving Eternal Lands players. Our guild is pretty quiet and mostly AFK these days. We are a democratic, non-PK guild that accepts members with an interest in Linux or Unix-like operating systems. You must have at least one skill (other than overall) above level 20 to join. Please contact a LNX player in-game to ask about membership.

Install the Eternal Land client by building it from the source code on GitHub or download builds from the EL website.
There are also new Flatpak and Snapcraft packages available.
